Selectores en CSS

Los selectores en CSS son patrones utilizados para seleccionar los elementos HTML que deseas estilizar. Existen varios tipos de selectores, entre los más comunes se encuentran:

  • Selector de tipo: Selecciona todos los elementos de un tipo específico. Ejemplo: p { color: blue; } selecciona todos los párrafos.
  • Selector de clase: Selecciona todos los elementos que tienen una clase específica. Ejemplo: .mi-clase { font-size: 16px; } selecciona todos los elementos con la clase "mi-clase".
  • Selector de ID: Selecciona un elemento único con un ID específico. Ejemplo: #mi-id { background-color: yellow; } selecciona el elemento con el ID "mi-id".
  • Selector universal: Selecciona todos los elementos en el documento. Ejemplo: * { margin: 0; padding: 0; } aplica a todos los elementos.
  • Selector de atributo: Selecciona elementos basados en un atributo o valor de atributo. Ejemplo: a[href^="https"] { color: green; } selecciona todos los enlaces que comienzan con "https".

Estos son solo algunos ejemplos de los muchos selectores disponibles en CSS. Utilizando estos selectores, puedes aplicar estilos específicos a diferentes partes de tu página web de manera eficiente.
